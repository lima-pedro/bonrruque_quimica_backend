# Backend BONRRUQUE QUÍMICA

## Todos endpoints possuem os seguintes recursos

- GET : Lista todos os dados do banco
- GET /id : Lista um dado específico
- POST : cria um dado
- PUT /id : atualiza um dado específico
- DELETE /id : apaga um dado específico

## Autenticação

É utilizado Json Web Token para validação de usuários e acesso à API.
